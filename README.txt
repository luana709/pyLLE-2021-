# pyLLE-2021 is an updated version of the open source pyLLE
https://github.com/gregmoille/pyLLE
https://usnistgov.github.io/pyLLE/

debugged for windows, python 3.9.7 and julia 1.5.4

once downloaded the original pyLLE library, copy paste the updated version of "llesolver.py" and "ComputeLLE.jl" in the Python folder
the Python folder should be somewhere like "C:\Users\XXX\AppData\Local\Programs\Python\Python39\Lib\site-packages\pyLLE"
